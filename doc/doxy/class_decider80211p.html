Calculates a SNR-Mapping for a Signal.

This method works as the calculateSnrMapping of the BaseDecider class, but it return the mapping for both SNR and SINR. This method is used to determine the frame reception probability for both SNR and SINR values. In this way we can determine (still probabilistically) if a frame has been dropped due to low signal power or due to a collision
