Detailed Description

Extended version of Two-Ray Ground path loss model.

See the Veins website for a tutorial, documentation, and publications .

An in-depth description of the model is available at: Christoph Sommer and Falko Dressler, "Using the Right Two-Ray Model? A Measurement based Evaluation of PHY Models in VANETs," Proceedings of 17th ACM International Conference on Mobile Computing and Networking (MobiCom 2011), Poster Session, Las Vegas, NV, September 2011.

Author
Stefan Joerer
